Image Hover Effects

Image hover effects are one of the many ways one can add a bit more flair to boring standard images in there html programs. As the MDN reference states "The hover CSS media feature can be used to test whether the user's primary input mechanism can hover over elements" and w3schools.com notes that "The CSS :hover pseudo-class is used to select elements when you mouse over them" and can be actually used on all elements and not just on links.
